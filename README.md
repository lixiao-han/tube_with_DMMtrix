# tube_with_DMMtrix
植株试管DM Matrix二维码图片与试管、试管架的结合（随机），并以此作为后续识别DM二维码的训练数据。因此，本方法为植株项目的数据增强模块，最终生成8位真彩图，用作后续的二维码识别定位。
具体流程可见项目任务流程图.vsdm
